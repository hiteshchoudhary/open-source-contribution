## repo refreshed

`Repo is refreshed` is a git repository for contributors. They can get a chance for their first contribution on gitHub by making a pull request to this repository. This is not limited to the freshers but everyone is welcomed.

## Getting Started

1. firstly, **fork** this repository
2. Secondly, **clone** this repository
```bash
git clone https://github.com/hiteshchoudhary/open-source-contribution
```
3. **Commit** your changes
4. Finally, make a **Pull Request (PR)** and wait for a while to get accepted.
