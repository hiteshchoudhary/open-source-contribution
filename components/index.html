Hello Everyone

Here is a Git code for beginners

Code Uses
git --version It is used for check version of git.
git status It is use for check status of your repo.Like it is untracked, modified or traked.
git init It is used for initialize git repo
git add It is used for add file in staging area
git commit -m "commit message" It is used committing the changes to the repository.The -m flag is used to add a message to the commit.
git log It will show you the history of your repo.
git log --oneline It will show only the commit message.
git branch This command lists all the branches in the current repository.
git branch bug-fix This command creates a new branch called bug-fix.
git switch bug-fix Used for switch to the bug-fix branch
git switch -c dark-mode This command creates a new branch called dark-mode. the -c flag is used to create a new branch.
git checkout bug-fix Used for switch to the bug-fix branch
git merge bug-fix This command merges the bug-fix branch into the main branch.
git branch -d bug-fix This command deletes the bug-fix branch.
git branch -m This command renames the old-branch-name to new-branch-name.
git remote add origin This command adds a remote repository to your local repository.
git push -u origin main This command pushes the changes to the remote repository.
git pull origin main This command pulls the changes from the remote repository.
git clone This command clones the remote repository to your local machine.
git diff This command shows the difference between the working directory and the staging area.
git diff --staged This command shows the difference between the staging area and the repository.
git diff This command shows the difference between two commits.
git reset This command unstages the file from the staging area.
git reset --soft This command resets the HEAD to the specified commit and keeps the changes in the staging area.
git reset --hard This command resets the HEAD to the specified commit and discards the changes in the working directory.
git checkout -- This command discards the changes in the working directory and reverts the file to the last commit.
git stash This command saves the changes in the working directory and reverts the file to the last commit.
git stash pop This command applies the changes saved in the stash to the working directory.
git stash list This command lists all the stashes.
git stash drop This command deletes the last stash.
git stash clear This command deletes all the stashes.
git tag This command creates a tag with the specified tag-name.
git tag -a -m "tag message" This command creates an annotated tag with the specified tag-name and tag message.
git tag This command lists all the tags.
git show This command shows the details of the specified tag.
git push origin This command pushes the specified tag to the remote repository.
