# GROUND RULES
#### Do not add new files or folders. Do not change `readme.md`, only add your name to `en/readme.md`.

## How to Contribute
1. Create your own fork of the code
2. Go to `en` folder
3. Add your name to the `en/readme.md` file
4. Commit
5. Raise Pull request
